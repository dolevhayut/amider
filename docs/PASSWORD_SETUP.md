## ❓ פתרון בעיות

### לא מצליח להגדיר סיסמה?
- ודא שיש לך הרשאות Admin בפרויקט
- נסה להתנתק ולהתחבר מחדש ל-Supabase Dashboard
- בדוק שהמייל נכון בדיוק

### משתמש לא מופיע ברשימה?
- ודא שהמשתמש נוצר ב-Supabase Auth
- אם לא - צור אותו ידנית:
  1. Authentication -> Users
  2. לחץ "Invite"
  3. הזן מייל
  4. המשתמש יקבל מייל להגדרת סיסמה

### סיסמה לא עובדת?
- נסה לאפס אותה שוב
- ודא שאין רווחים בתחילה/סוף
- נסה סיסמה אחרת זמנית
