## מה שונה? 🔄

### 1. פונט דתי "Lia Batkol" ✍️
- **הוסף ל**: `src/index.css`
- **משתמש ב**: כל הכותרות בדף הנחיתה ועמוד התרומה
- **אפקט**: מראה רוחני ומכובד יותר

**לפני**:
```

```

**אחרי**:
```

```

---

### 2. רקע צדיקים 🖼️
- **קובץ**: `פתיח-למובייל-3.jpg`
- **מיקום**: רקע של הירו סקשן
- **אטימות**: 10% (לא מפריע לתוכן)
- **Overlay**: גרדיאנט לבן מלמעלה למטה

**תוצאה**: אווירה רוחנית ומרגשת בלי להפריע לקריאות

---

### 3. לוגו המיזם 🏛️

**מופיע ב**:
- ראש דף הנחיתה (גדול)
- ראש עמוד תרומה (בינוני)
- פוטר של שני הדפים (קטן)

**גדלים**:
- דף נחיתה: `h-16 sm:h-20 md:h-24`
- עמוד תרומה: `h-12`
- פוטר: `h-12` עם אטימות 60%

### 4. אלמנטים דקורטיביים ✨

---

### 5. אנימציות כניסה 🎬

**סוגי אנימציות**:
- `fadeInUp` - כניסה מלמטה עם fade
- `fadeIn` - הופעה הדרגתית

**סדר הכניסה** (בהירו סקשן):
1. כותרת ראשית (0ms)
2. כותרת משנה (100ms)
3. תיאור (200ms)
4. אלמנט דקורטיבי (300ms)
5. כפתורי CTA (400ms)

**משך**: 0.6-0.8 שניות
**אפקט**: smooth ומקצועי

## ביצועים ⚡

### גודל הנכסים:
- **פונט WOFF**: ~50KB (טעינה מהירה)
- **לוגו PNG**: ~20KB (ממוטב)
- **רקע JPG**: ~150KB (דחוס)

### אופטימיזציות:
- `font-display: swap` - טקסט נראה מיד
- אטימות נמוכה לרקע - פחות עומס ויזואלי
- אנימציות CSS (לא JS) - ביצועים מעולים
- גדלים responsive - טוען רק מה שצריך
