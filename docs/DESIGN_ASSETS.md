# נכסי עיצוב - דף נחיתה 🎨

## קבצים שנוספו

### 1. פונט דתי - Lia Batkol
**קובץ**: `/public/LiaBatkol-regular.woff`

הפונט הדתי משמש לכל הכותרות בדף הנחיתה ועמוד התרומה.

**איך להשתמש**:
```css
/* הפונט כבר מוגדר ב-src/index.css */
@font-face {
  font-family: 'Lia Batkol';
  src: url('/LiaBatkol-regular.woff') format('woff');
}

/* השתמש בקלאס */
.landing-page-title {
  font-family: 'Lia Batkol', 'Google Sans', serif;
}
```

**איפה משתמשים**:
- כותרת ראשית בדף נחיתה
- כותרות משנה
- כותרות סקשנים (אודות, השפעה, וכו')
- כותרת עמוד תרומה

---

### 2. תמונת רקע - צדיקים
**קובץ**: `/public/פתיח-למובייל-3.jpg`

תמונת רקע רוחנית המופיעה בהירו סקשן של דף הנחיתה.

**איך זה עובד**:
```jsx
// ב-LandingPage.tsx
<div 
  className="absolute inset-0 opacity-10"
  style={{
    backgroundImage: "url('/פתיח-למובייל-3.jpg')",
    backgroundSize: 'cover',
    backgroundPosition: 'center'
  }}
/>
```

**מאפיינים**:
- אטימות נמוכה (10%) - לא מפריע לתוכן
- מכסה את כל שטח ההירו
- יש overlay לבן שמוסיף עומק

---

### 3. לוגו המיזם
**קבצים**: 
- `/public/logo.png`
- `/public/לוגו-עמידר-לאתר-1.png`

הלוגו מופיע במספר מקומות:

#### דף נחיתה:
- בראש הדף (מעל הכותרת)
- בפוטר (קטן יותר)

#### עמוד תרומה:
- בראש הדף
- בפוטר

**גדלים**:
```jsx
// ראש דף נחיתה
<img src="/logo.png" className="h-16 sm:h-20 md:h-24" />

// עמוד תרומה
<img src="/logo.png" className="h-12" />

// פוטר
<img src="/logo.png" className="h-12 opacity-60" />
```

## CSS Classes החדשים

### `.landing-page-title`
```css
.landing-page-title {
  font-family: 'Lia Batkol', 'Google Sans', serif;
}
```

**איפה להשתמש**:
```jsx
<h1 className="landing-page-title text-4xl font-bold">
  כותרת בפונט דתי
</h1>
```

### אנימציות כניסה

#### `.animate-fade-in-up`
אנימציית fade in + תזוזה מלמטה למעלה

```jsx
<h1 className="animate-fade-in-up">
  כותרת שנכנסת מלמטה
</h1>
```

#### `.animate-fade-in`
אנימציית fade in פשוטה

```jsx
<div className="animate-fade-in">
  תוכן שמופיע בהדרגה
</div>
```

#### עיכובים (Delays)
```jsx
<div className="animate-fade-in-up animate-delay-100">100ms</div>
<div className="animate-fade-in-up animate-delay-200">200ms</div>
<div className="animate-fade-in-up animate-delay-300">300ms</div>
<div className="animate-fade-in-up animate-delay-400">400ms</div>
```

**סדר האנימציות בדף הנחיתה**:
1. כותרת ראשית (0ms)
2. כותרת משנה (100ms)
3. תיאור (200ms)
4. אלמנט דקורטיבי (300ms)
5. כפתורי CTA (400ms)
