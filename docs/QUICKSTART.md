## 📱 דפים במערכת

### Dashboard (`/messenger/dashboard`)
- סטטיסטיקות כלליות
- כלי שיווק (קישור + QR)
- רשימת תורמים מקוצרת
- ניהול ארנק

### ניהול תורמים (`/messenger/donors`)
- טבלת תורמים מלאה
- חיפוש וסינון
- סטטיסטיקות מפורטות

### ניהול תפילות (`/messenger/prayers`)
- רשימת כל התפילות
- סימון תפילות כהושלמו
- סינון לפי סטטוס

### הגדרות (`/messenger/settings`)
- עדכון פרטים אישיים
- הגדרות דף נחיתה
- מידע על התוכנית

## 🎨 עיצוב וחווית משתמש

- **צבעים:** Indigo כצבע ראשי
- **כיוון:** RTL (עברית)
- **רספונסיבי:** תומך במובייל, טאבלט ודסקטופ
- **אייקונים:** Lucide React
- **סטייל:** Tailwind CSS
